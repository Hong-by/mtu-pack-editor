MTU Pack Editor reference pack folder
====================================

Put reference packs here when you want the editor to expose vanilla, DLC,
and companion-mod character data.

Default reference file names:

  BFG_Originals.pack
  database.pack
  data_mh.pack
  data_ep.pack
  data_dlc07.pack
  data_dlc06.pack
  data_bl.pack
  data_yt_bl.pack
  data.pack

Missing reference packs are reported in the UI. The primary input pack
still belongs one level up as:

  ..\my_hero.pack
