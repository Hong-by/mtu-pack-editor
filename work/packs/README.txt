MTU Pack Editor pack folder
===========================

Put the primary MTU mod pack here before running the editor.

Required default file:

  my_hero.pack

The generated patch pack is saved here by default:

  my_hero_patch.pack

Reference packs go in the refs folder.

Public GitHub releases do not include Total War: THREE KINGDOMS game data
or third-party mod pack files. Use pack files that you already own or have
permission to use.
